Steps for running Selenium Tests
1. Open selenium-tests as a project in Eclipse
2. Open file selenium-tests/src/framework/GroupSevenTests.java
3. Add the client-combined jar files in selenium-java-3.7.1 as external jars to the project
4. Add all files in selenium-java-3.7.1/libs as external jars to the project
5. Run the app (instructions are in l02-07/setup.txt)
5. Run GroupSevemTests.java
